Créer un projet

PSI permet de creer un projet à partir d'un fichier Open Workbench.
Open Worbench est une application bureautique permettant la planification et la gestion de
projet. Il s'agit d'un logiciel libre, correspondant à une alternative de Miscrosoft Project
(pour plus d'information : http://www.openworkbench.org/).
Les informations telles que le nom du projet ainsi que les listes des ressources sont
récupérées. Procédure pour créer un projet :

  • Cliquer sur fichier
  • Cliquer sur creer un projet
  • Selectionner le fichier open Workbench (.xml)
